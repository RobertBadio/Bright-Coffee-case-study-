Bright-Coffee-case-study
Coffee Shop Sales Analysis for the manager
Project Overview
my project analyzes transactional coffee shop data to uncover insights that help improve revenue and product performance for the coffee shop.
The goal is to transform raw sales data into meaningful business intelligence that supports better decision-making for coffee shop management.

Objectives
- Identify top-performing products highest 
- Analyze peak sales times
- Evaluate sales trends
- Provide business recommendations on productivity

Tools to Used
- SQL (Databricks) – Data querying and transformation
- Microsoft Excel – Initial data handling and visualization
- Power BI / Excel – Dashboard creation
- Microsoft PowerPoint – Final presentation
- Miro – Data architecture and planning
- Canva – Visual design
  
Data Processing
- Converted Excel files to CSV to upload
- Converted Excel files to CSV format
- Cleaned and structured data
- Uploaded into Databricks
- Created time buckets (minutes hours intervals)
- Calculated total_amount = unit_price * quantity
- Performed SQL queries to extract insights
- Aggregated sales by product, category, and time
- Created a PowerPoint report for stakeholders

Key Insights
- Best-selling products
- Peak hours of sales
- Revenue contribution per category
- Identified best-selling products driving revenue
- Peak hours showed highest transaction volumes
- Off-peak periods revealed opportunities for promotions
- Sales patterns vary by time of day
- Some products generate high volume but low margin

Recommendations
-Introduce bundle deals to increase average spend
-Optimize staffing during peak hours
-Improve performance of low-performing products
-Replicate strategies from top-performing store
-Introduce promotions during off-peak hours
