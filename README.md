# Bright-Coffee-case-study-
Coffee Shop Sales Analysis for the manager
## Project Overview
my project analyzes transactional coffee shop data to uncover insights that help improve revenue and product performance for the coffee shop.

## Objectives
- Identify top-performing products highest 
- Analyze peak sales times
- Evaluate sales trends
- Provide business recommendations on productivity

## Tools to Used
- SQL in databricks 
- Excel and Power BI
- Miro mind map design
  
## Data Processing
- Converted Excel files to CSV to upload
- Created time buckets (3 hours intervals)
- Calculated total_amount = unit_price * quantity

## Key Insights
- Best-selling products
- Peak hours of sales
- Revenue contribution per category

## Recommendations
- Increase stock of high-performing items
- Run promotions during slow hours
- Improve visibility of low-performing products

## Next Steps
- Automate the reporting
- Expand to multi-location tracking units

  ## Data Architecture and design
